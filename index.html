In this assignment I used HTML (Hypertext Markup Language, CSS (Cascading Style Sheets), and Javascript to build this site. HTML is used as the backbone of this site, as it contains the text and images used in this site. CSS is used to make the HTML elements look better, as it offers the tools to change things such as size, font, and where things are placed on the site. Javascript is used to add interactivity to HTML elements, allowing you to interact with some elements on this site such as the image gallery.
